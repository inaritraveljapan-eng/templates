**Стандарты программы:**
- Times New Roman, бежевые заголовки дней
- Названия объектов: сначала по-русски, в скобках — оригинал (японский/английский)
- Текст выровнен по ширине, без курсива (только bold для названий)
- Нет цен в программе
- Meet & Greet в аэропорту — описать словами, не включать в расчёт
- В конце программы: блок контактов INARI TRAVEL

### 滞在予定表 (Itinerary для японского консульства)

**Формат документа:**
- A4 портрет, Meiryo UI
- Заголовок: `Mr. [Имя] その他[N]名滞在予定表は記の通りです。`

- Колонки таблицы: 到着予定日 | 出発予定日 | 宿泊予定先 | 住所・電話番号 | 連絡先

- Названия отелей в 宿泊予定先: катакана/кандзи + адрес по-японски

**Формат ваучеров (DOCX):**
- Номер ваучера: YYMMNN-[порядковый]

- Строку Booking Number удалять если нет номера подтверждения
- Footer: Century Gothic, #C00000, INARI TRAVEL + адрес + тел.
